# Quiz Biblique Game

## Project Documentation

### Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Abdiassino/quiz-biblique.git
   cd quiz-biblique
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

### Project Structure
- `src/` - Contains the source code for the game.
- `public/` - Contains static assets such as images and stylesheets.
- `tests/` - Contains unit and integration tests for the project.
- `README.md` - This file with project documentation.

### Features
- Interactive quiz experience based on biblical questions.
- Scoring system to track user performance.
- Timer functionality to enhance challenge.
- High scores leaderboard to encourage competitiveness.

### API Overview
- **GET /api/questions** - Fetch a list of quiz questions.
- **POST /api/score** - Submit a user's score after completing a quiz.
- **GET /api/scores** - Retrieve high scores from the leaderboard.
